card option code input 5

cout carry out output 1
zero flag of zero output 1

Name Description Option Code
ADD add without carry 5'b00000
ADDC add with carry 5'b00001
SUB sub without carry 5'b00010
SUBC sub with carry 5'b00011
SUBR sub reversed without carry 5'b00100
SUBCR sub reversed with carry 5'b00101
ASA assign F with A 5'b00110
ASB assign F with B 5'b00111
RSA reverse a 5'b01000
RSB reverse b 5'b01001
OR logical add 5'b01010
AND logical multiplication 5'b01011
NOR not exclusive or 5'b01100
XOR exclusive or 5'b01101
NAND reversed logical multiplication 5'b01110
ASZ assign F with 0 5'b01111

Achieve

  • 示例 https://www.cnblogs.com/liutianchen/p/7616754.html

  • 注:这个代码写的很烂,但是还是能看清楚大致的思路

  • 进位可以使用连接符进行获取,设 result 为 32 bits,操作数 op1 和 op2 为 32 bits,进位标识符 cout 为 1bit 

    //带进位加法
{cout,result} = op1 + op2;

    连接符表示 cout 和 result 合成一个 33 位的变量,如果 32 加法溢出,溢出会进位到第 33 位,即 cout 位,减法同理

  • 为了观察到溢出,测试用例可以使用边界的情况,如 a + b + cin = 32'b11…10 + 1 + 1 = {1,32'b0} = {cout,result}

  • verilog 中～表示按位取反，！表示逻辑取反，逻辑运算应当使用！

  • 注意算术运算和逻辑运算的优先级，如果不能确定优先级请 使用小括号
